# Home Assistant App: LedFx

## How to use

Install the LedFx app from this repository and start it from the Home Assistant Apps page.


LedFx can be accessed on port `8888` of you host (HAOS), for example ``92.168.1.5:8888`.

## Notes

- This app uses the upstream Docker image from `ledfxorg/ledfx`.
- No additional configuration is required for the web UI.
- If you want to publish this app repository, update the app image to your own container registry as needed.
